# Business Insights

Este documento reúne alguns insights obtidos a partir das análises realizadas no banco de dados AgroAnalyticsDB.

O objetivo deste documento é demonstrar como consultas SQL podem gerar informações úteis para apoiar decisões no setor agrícola.

## Receita por Fazenda
A análise da receita total por fazenda permite identificar quais propriedades apresentam maior capacidade de geração de faturamento.

Consulta utilizada:
queries/05_metricas_financeiras.sql

Indicador analisado:
- Receita total por fazenda.

## Fazendas Mais Rentáveis
A comparação entre custos de produção e receita permite identificar as fazendas com melhor desempenho financeiro.

Indicadores analisados:
- Receita total.
- Custo total.
- Lucro bruto.
- ROI.

## Desempenho das Culturas
As análises por cultura permitem identificar quais produtos agrícolas apresentam maior produção e maior potencial de geração de receita.

Indicadores analisados:
- Produção total.
- Receita total.
- Produtividade média.

## Investimento em Máquinas
A análise dos investimentos em maquinário permite avaliar o nível de capital empregado em cada fazenda.

Indicadores analisados:
- Quantidade de máquinas.
- Valor total investido.

## Principais Compradores
A análise dos compradores permite identificar os clientes responsáveis pelo maior volume financeiro das vendas.

Indicadores analisados:
- Faturamento por comprador.
- Ranking de compradores.

## Indicadores Climáticos
Os dados climáticos permitem acompanhar fatores ambientais que podem impactar a produtividade agrícola.

Indicadores analisados:
- Temperatura média.
- Precipitação média.
